1.启动项目
django-admin startproject opsweb

	(python36env) [wzj@localhost opsweb]$ tree 
	.
	├── manage.py  		命令行工具
	└── opsweb	   
	├── __init__.py 	python模块
	├── settings.py  	项目配置文件

	└── wsgi.py 		WSGI兼容性web服务器入口

3.启动服务
	配置mysql：
		DATABASES = {
	     'default': {
	         'ENGINE': 'django.db.backends.mysql',
	          'NAME': 'django',
	          'USER': 'root',
	          'PASSWORD': '123456',
	          'HOST': '127.0.0.1',
	          'PORT': 3306,
	      }
	    }
	编辑__init__.py 
		import pymysql
		pymysql.install_as_MySQLdb()
	启动服务器
		python manage.py  runserver 

4.创建一个app
	python manage.py startapp dashboard
  配置url
  	from django.conf.urls import include, url
  	url(r'^dashboard/', include("dashboard.urls")),
  激活app
